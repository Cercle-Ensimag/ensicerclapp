Fonctionnalités

Actualités

Consulte les actualités publiées par les différentes associations, notamment par le Pole Com'.

Cafet

Consulte ton solde cafet et passe commande. (Pas encore disponible)

Calendrier

Ce calendrier regroupe tes cours, les événements associatif que tu y as ajouté, et tes événements personnels.
Un événement en cours est affiché en vert

Pour ajouter ou retirer un événement associatif, réfère-toi à la Fonctionnalité 'Evénements'.

Les cours sont téléchargés en fonction des sessources données dans le champ 'Ressources ADE' des paramètres du calendrier settings.
Pour obtenir ces ressources, connecte-toi à ADE depuis Zenith et copie les ressources présentes dans l'URL.
Il faudra mettre à jour ces ressources à chaque fois que tu t'incrit à un nouveau cours, c'est-à-dire en général au début de chaque semestre.

Pour créer un événement personnel, utilise l'icone add_circle et utilise edit pour le modifier. Pour quitter le mode édition clique sur close
Le contenu de ces événement peut pour l'instant être consulté par l'administrateur.

Evénements

Consulte les événements créés par les associations de l'Ensimag
Pour ajouter un événement à ton calendrier, clique sur event_available et clique sur event_busy pour l'en retirer.

Vote

Vote pour une élection ou un sondage. Ton vote est totalement anonyme en base de donnée.

Mentions légales

L'application web Cercle Ensimag (nom de code : ensicerclapp) propose un ensemble de fonctionnalités aux élèves de l'Ensimag qui visent à soutenir la vie étudiante au sein de l'école.

Le code de l'application est disponible sur GitHub.

Données personnelles

Les données personnelles collectées par le Cercle Ensimag sont strictement nécessaires à la bonne execution des services proposés.
Les données qui n'impactent pas la bonne execution des services sont supprimées à la suppression du compte, les autres -- tel que les registres de vote -- sont conservées jusqu'à ce qu'elles ne soient plus nécessaires.

La date de certaines actions sert d'identifiant unique à des données mises en base de données mais aucun traitement n'est effectué sur celle-ci.

Des données anonymes de consultation sont récupérées via Google Analitics pour obtenir des statistiques sur l'utilisation de l'application.
